### Cài đặt trên local Window OS

Tạo thư mục dự án

```sh
mkdir my_project && cd my_project
```

Cài và setup môi trường ảo

```sh
pip install virtualenv
py -m venv env
cd env/Scripts
activate
pip install flask requests opencv-python json
cd..
cd..
```

Clone code và chạy trên local

```sh
git clone https://github.com/tranthuc99/project2.git
cd project2
py app.py
```

Chạy url: http://127.0.0.1:5000/ trên trình duyệt web!
